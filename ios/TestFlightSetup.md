## Solace-Mobile-App - TestFlight Setup

This document outlines the process for setting up TestFlight to install and test the Solace-Mobile-App on iOS.


[](HOSTING SOLACE-MOBILE-APP ON TESTFLIGHT)
1. Setup an Apple Developer Account.
  - Go to [Apple Developer](https://developer.apple.com/) and sign up.
2. Create a new app project in Xcode.
  - Open Xcode and create a new project.
  - Select "Single&quot;file' when asked.
  - Use the option to create a new team.
3. Set up TestFlight for private testing.
  - G[É to the Apple Developer portal and select TestFlight.
  - Add testers using their Apple ID.
4. Invite testers to try the app.
  - Send an invitation link to testers so they can install the app.
